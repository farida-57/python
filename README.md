3.Project Description:
    This program is a console-based order calculator for Sunrise Coffee. It allows the user to enter customer details, select quantities of coffee, tea, and sandwiches, and input unit prices. The program calculates the total cost, applies a manual discount, and displays a clear summary of the order.
